## Инструкции для создания

Изображения могут быть созданы с помощью:

1. **Диаграммы состояний**:
   - Draw.io / Lucidchart для графов состояний
   - PlantUML для автоматической генерации
   - Graphviz для программной генерации

2. **Временные диаграммы**:
   - GTKWave для экспорта из симуляций
   - WaveJSON для веб-генерации
   - Python matplotlib для скриптового создания

3. **Структурные схемы**:
   - Inkscape для векторной графики
   - Vivado Schematic Viewer для экспорта схем
   - Draw.io для блок-диаграмм

4. **Алгоритмические диаграммы**:
   - LaTeX + TikZ для математических диаграмм
   - Python + matplotlib для пошаговых визуализаций
   - PowerPoint/Keynote для презентационных схем

## Рекомендуемые размеры
- **Ширина**: 1200-1600 пикселей
- **Высота**: 800-1200 пикселей
- **Формат**: PNG для диаграмм, SVG для масштабируемости
- **Разрешение**: 150-300 DPI для четкости

## Цветовая схема
- **Состояния**: Светло-голубой (#E3F2FD)
- **Переходы**: Темно-синий (#1976D2)
- **Входы**: Зеленый (#4CAF50)
- **Выходы**: Оранжевый (#FF9800)
- **Ошибки**: Красный (#F44336)
- **Текст**: Черный (#212121) или темно-серый (#424242)

## Автоматическая генерация

Для автоматической генерации диаграмм из кода можно использовать:

```bash
# Из SystemVerilog в диаграмму состояний
sv2state_diagram traffic_light.sv > traffic_light_fsm.dot
dot -Tpng traffic_light_fsm.dot -o chapter_4_traffic_light_states.png

# Из simulation в временные диаграммы
gtkwave dump.vcd --vcd --png --output chapter_4_timing.png

# Генерация схем из Vivado
vivado -mode batch -source export_schematic.tcl
```

## Примеры инструментов

### Для диаграмм состояний:
```python
# Python + graphviz для автоматической генерации
import graphviz

fsm = graphviz.Digraph('finite_state_machine', comment='Traffic Light FSM')
fsm.node('IDLE', 'IDLE\n(Initialize)')
fsm.node('UD_GREEN', 'UD_GREEN\n(Up-Down: Green)')
fsm.edge('IDLE', 'UD_GREEN', 'timer_expires')
```
